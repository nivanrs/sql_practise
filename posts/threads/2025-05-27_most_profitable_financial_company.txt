[1/9]
Kalau kamu hanya cari perusahaan finansial dengan profit tertinggi secara global, kamu tidak tahu siapa yang memimpin di setiap benua. Peringkat global bisa didominasi satu region dan menyembunyikan pemain kuat di region lainnya yang punya kondisi pasar berbeda.

[2/9]
Dari dataset Forbes Global 2010-2014, temukan perusahaan finansial paling menguntungkan di dunia. Lalu, siapa yang memimpin di masing-masing benua? Soal ini melatih cara memanfaatkan rank yang sudah ada di tabel dan cara membuat ranking baru dengan window function.

[3/9]
Query global: rank sudah ada di tabel, tidak perlu dihitung ulang.
```sql
SELECT company, continent
FROM forbes_global_2010_2014
WHERE sector = 'Financials'
  AND rank = 1;
```

[4/9]
Query per benua: ROW_NUMBER() membuat ranking baru per kelompok benua.
```sql
WITH ranked AS (
  SELECT company, continent,
    ROW_NUMBER() OVER (
      PARTITION BY continent
      ORDER BY profits DESC
    ) AS rn
  FROM forbes_global_2010_2014
  WHERE sector = 'Financials'
)
SELECT company, continent FROM ranked WHERE rn = 1;
```

[5/9]
Kenapa query global cukup pakai WHERE rank = 1?

Karena kolom rank sudah tersedia di tabel dan sudah mencerminkan peringkat profit global. Tidak perlu menghitung ulang dengan window function. Membaca struktur tabel sebelum menulis query menghemat banyak kode yang tidak perlu ditulis.

[6/9]
Kenapa query per benua butuh ROW_NUMBER() OVER (PARTITION BY continent)?

Karena ranking global tidak bisa dipecah per benua secara langsung. ROW_NUMBER() membuat ulang peringkat dalam setiap partisi benua secara terpisah dan independen. Tanpa PARTITION BY, semua baris mendapat nomor urut global yang tidak berguna untuk perbandingan antar region.

[7/9]
Kenapa memakai CTE (WITH ranked AS ...) dan bukan subquery bertumpuk?

CTE memisahkan logika windowing dari logika filtering. Query utama menjadi bersih: hanya ambil baris dengan rn = 1. Subquery bertumpuk di dalam FROM membuat query lebih sulit dibaca, lebih sulit di-debug, dan lebih sulit dimodifikasi saat kebutuhan berubah.

[8/9]
Tim strategi bank global memakai analisis ini untuk membandingkan posisi kompetitif di setiap region. Pemimpin profit global belum tentu mendominasi setiap benua karena faktor regulasi, infrastruktur keuangan, dan dinamika pasar berbeda secara signifikan di setiap region.
